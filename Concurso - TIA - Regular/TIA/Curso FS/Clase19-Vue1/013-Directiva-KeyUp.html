Vue.js: Directiva key.up

Vue.js ofrece modificadores de teclas mediante los cuales se manejan los eventos. Si necesitamos que un input llame a un método definido en una instancia Vue sólo cuando se pulsa/suelta la tecla enter podemos agregar un modificador como el siguiente:

<input type = "text" v-on:keyup.enter = "...."/>

Esto funciona con prácticamente cualquier tecla.
