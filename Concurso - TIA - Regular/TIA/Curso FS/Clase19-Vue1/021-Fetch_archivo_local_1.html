Fetch (Archivo) - Sin usar VUE

En este ejemplo leemos un archivo local, utilizando fetch, pero sin usar Vue.

La API fetch permite acceder a recursos de manera asíncrona. Para utilizar esta API no es necesario usar ninguna librería. Funcionan haciendo uso del código nativo del navegador.

La API fetch utiliza promesas, es decir, devuelve un objeto con dos métodos, uno then() y otro catch() a los que podemos pasar una función que se invoca cuando se obtiene una respuesta a esa promesa o se produce un error.
