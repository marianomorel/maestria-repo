Arrays (Arreglos o vectores)

Un array, también conocido como arreglo o vector, es una colección o agrupación de elementos en una misma variable. Los elementos del array pueden ser datos de diferentes tipos. Sin embargo, algunos de los métodos que poseen sólo funcionarán correctamente en arrays que tengan todos sus elementos del mismo tipo. Cada elemento dentro del array posee un índice, un valor que nos permite identificarlo. Pensábamos a las variables como una “caja”. De forma similar, podemos imaginar un array como los vagones de un tren, donde cada vagón posee un contenido y un orden. El índice es el orden y el contenido dentro del vagón es el dato.
